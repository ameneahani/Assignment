# Palam Polom ✋🤚

A simple hand game built with **Python**.

---

## 📌 Features
- Play against two computer players.
- Choose between ✋ (1) and 🤚 (2).
- Score is calculated based on the game rules.
-  Reset and New Game functionality.

---

## 🛠️ Requirements
- Python 3.x

---

```bash
python palam_polom_game.py
