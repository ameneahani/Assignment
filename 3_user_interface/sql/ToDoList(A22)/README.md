# To-Do List App (PySide6 + SQLite)

A simple desktop To-Do List application built with **PySide6** (Qt for Python) and **SQLite**.  
You can add, edit, mark as done, and delete tasks. Each task can have a title, description, and time.

---

## âœ¨ Features
- Add new tasks with title, description, and time
- Mark tasks as done / undone
- Delete tasks
- Tasks marked as done are displayed first
- Tasks with priority are shown in red
- Data stored in an **SQLite** database (`todolist.db`)
