# Python Classes Project

This repository contains several Python classes and programs for learning and practicing object-oriented programming. The main files are:

- `11_1_fraction_class.py`: Implements the `Fraction` class to work with fractions.
- `11_2_time_class.py`: Implements the `Time` class to work with hours, minutes, and seconds.
- `11_3_complex_class.py`: Implements the `Complex` class to work with complex number.
- `11_4_rug_pattern.py`: A main program that uses the classes and demonstrates their functionality.
