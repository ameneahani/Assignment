Pong Game 2025 🏓

A simple Pong game built with Python and the Arcade library.

📌 Features

Classic Pong gameplay

Player vs AI (computer opponent)

Score tracking system

Smooth paddle & ball movement

Collision detection

🛠️ Requirements

Python 3.8+

arcade
 library

Install Arcade:

pip install arcade

▶️ Run

Clone the repo and run:

python pong.py

🎮 Controls

Arrow Keys (↑ / ↓) → Move left paddle (Player 1)

Right paddle is controlled by AI

Ball bounces automatically
